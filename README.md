# kevlar-encryption
Encryption Software

# To Install
-git clone <link>
-cd into repo directory 
-run the following:
$ make

# Documentation
Kevlar Encryption
    Create Keys     |   -c <key-names>
    Encrypt File    |   -e <plaintext> <public-key> <outfile-name>
    Decrypt File    |   -d <encrypted-file> <private-key> <outfile-name>
    Help            |   -h Prints This Message
